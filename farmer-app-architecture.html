FarmDirect Marketplace Architecture

Mobile-First Farmers Market on Google Cloud Platform

Backend Logic

  • • Cloud Run hosts the primary API (Express or Go).
  • • Handles logic for harvest listing creation and quantity tracking.
  • • Firestore Triggers handle realtime price updates for customers.

Networking & Infra

  • • Firebase Identity manages secure logins for Farmers/Buyers.
  • • Global LB with Cloud Armor protects against DDoS/API abuse.
  • • Pub/Sub decouples payments and notification logic (SMS/Email).

Data Modeling

  • • Listings: Type, Weight, Price, Expiry, Farmer Ref.
  • • Orders: Buyer ID, Farmer ID, Item ID, Qty, Status.
  • • Cloud Storage stores high-res harvest photos for buyer browsing.
